This page is for anyone who wishes to use see the Json Objects

If you want to see the whole lists of Surah and their information:

https://wacky-believe-production.up.railway.app/api/surahList

If you want to see only a specific Surah with just arabic text, please enter the index/Surah number here:

https://wacky-believe-production.up.railway.app//arabic/{number}

If you want to see only a specific Surah with translated text, please enter the index/Surah number and language from the list of languages here:

https://wacky-believe-production.up.railway.app//otherLang/{number}/{language}

If you want to see only a specific Surah with romanized text, please enter the index/Surah number and language here:

https://wacky-believe-production.up.railway.app//romanized/{number}

If you want to see a specific saved Update Progress using an email:

https://wacky-believe-production.up.railway.app/api/load/{email}
